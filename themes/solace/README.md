# Solace — Funeral Directors HTML Template

A calm, dignified, responsive HTML + Tailwind CSS template for funeral
directors and bereavement services, shipped with a complete demo firm
(Fairbourne & Sons Funeral Directors, Harrogate — family-owned since
1932). Five pages: Home, Services, Our promises, About and Contact —
built around what grieving families actually need: a 24-hour phone line
everywhere, a plain "what to do first" guide, gentle language and
transparent fees.

**Pricing transparency:** the CMA requires UK funeral directors to
display a standardised price list. The template links to "Our full
standardised price list" from the home page and every footer — point
those placeholder links at your own price list page or PDF before going
live.

**No build step is required** — the compiled CSS is included. Node.js is
only needed to change colours/fonts (edit `src/input.css`, then
`npm install && npm run build`). Every colour is a CSS variable; Solace
uses a deep dusk primary with a soft brass accent.

Editing: find-and-replace **Fairbourne & Sons**, **01423 552 132** and
**fairbourneandsons.co.uk**; swap the photos in `assets/img/` for your own
premises, chapel of rest and grounds (keep width/height attributes);
paste the kind words real families have sent you over the demo ones.
Forms use a demo handler (`data-demo-form` in `assets/js/main.js`) —
point them at Formspree/Netlify Forms to go live.

Tested at iPhone widths (390/375px, no horizontal scroll), WCAG-AA
patterns throughout, apple-touch-icon included. Demo photographs from
[Unsplash](https://unsplash.com/license); all firms, people, tributes
and numbers are fictional placeholder content.
