# Swim — Day-Ticket Fishing Lakes / Fishery HTML Template

A premium, responsive HTML + Tailwind CSS template for day-ticket
fisheries and coarse fishing lakes, shipped here with a complete demo
venue (Willow Pool Fisheries, Shropshire — three lakes on a family farm,
open since 1987). Five pages: Home, Lakes & Prices, Catch Reports, About
and a find-us/contact page — built around what anglers actually check
before travelling: prices, opening times, the rules, recent catches and
a phone number everywhere.

**No build step is required** — the compiled CSS is included. Node.js is
only needed to change colours/fonts (edit `src/input.css`, then
`npm install && npm run build`). Every colour is a CSS variable.

Editing: find-and-replace **Willow Pool Fisheries**, **01939 260 410** and
**willowpoolfisheries.co.uk**; swap the photos in `assets/img/` for your own
lake and catch photos (keep width/height attributes); paste your real
catch reports and reviews over the demo ones. Forms use a demo handler
(`data-demo-form` in `assets/js/main.js`) — point them at
Formspree/Netlify Forms to go live.

Tested at iPhone widths (390/375px, no horizontal scroll), WCAG-AA
patterns throughout, apple-touch-icon included. Demo photographs from
[Unsplash](https://unsplash.com/license); all venues, people, catches
and numbers are fictional placeholder content.
