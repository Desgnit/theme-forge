# ThemeForest submission kit

ThemeForest (Envato) is human-reviewed and rejects near-duplicate items, so
per the strategy in `marketing/listings.md`:

1. Submit **Forgeline** first (most distinct), then **Terrace as a single
   "10 colour editions included" item** — do NOT list the ten football
   editions separately here.
2. Add **Sideline** once the first two are approved.
3. Keep the per-edition listings for Gumroad/Etsy/Creative Market.

## What Envato requires per item

| Requirement | Where to get it |
|---|---|
| Main file zip | `marketing/dist/<slug>-theme-v1.0.0.zip` (for the Terrace bundle, zip all ten `themes/` edition folders together) |
| Thumbnail 80×80 | `marketing/marketplaces/assets/themeforest/<slug>-thumbnail-80x80.png` |
| Inline preview 590×300 | `marketing/marketplaces/assets/themeforest/<slug>-preview-590x300.png` |
| Screenshots | `marketing/covers/*.png` (add per-page screenshots if the reviewer asks) |
| Live preview URL | the demo URLs on https://desgnit.github.io/theme-forge/ |
| Documentation | each theme ships a README — Envato reviewers expect an HTML or PDF docs folder; convert the README before submitting |

## Categories

- Forgeline → Site Templates → Corporate
- Terrace (bundle) & Sideline → Site Templates → Nonprofit

Item descriptions: use the corresponding files in this directory
(`forgeline.md`, `terrace-bundle.md`, `sideline.md`) — Envato's editor
accepts limited HTML; paste the text and re-apply bold/lists there.
