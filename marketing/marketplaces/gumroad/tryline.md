## Summary (one-liner)

A complete rugby league club website in deep maroon and amber — fixtures, league table with points difference, squad by position, juniors and match reports, with a gritty condensed-type design of its own.

## Description

Built for the community game — the demo club (Millbrook Raiders RLFC, est. 1923, The Foundry Ground, NCL Division Two) reads like a proper northern club, not a reskin.

- **Match centre**: rugby league scorelines, W/L/D badges, league table with points difference, all plain HTML edited by copy-paste
- **Squad grid** by position group (props to loose forwards) with player pages showing tries, goals and appearances
- Juniors page from tag rugby U8s to U16s with parents' FAQ
- Match report layout, club history, clubhouse news, contact with ground directions
- Its own design identity: Barlow Condensed display type, sharp corners, diagonal-stripe hero treatment
- 8 pages, HTML + compiled Tailwind CSS, no build step, vanilla JS, WCAG-AA, iPhone-tested
- Every colour is a CSS variable — five-minute rebrand, documented in the README
