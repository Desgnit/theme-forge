## Description

Anvil is an original, from-scratch Shopify OS 2.0 theme that builds in the features stores usually pay monthly apps for:

- **Mega menu** with image blocks, predictive-search-ready header, cart bubble
- **Quick-view** product modals straight from collection grids
- **Slide-out AJAX cart** with a free-shipping progress bar
- **Sticky add-to-cart** bar on product pages, dynamic checkout buttons, variant-aware pricing with sale badges
- Promo tiles, countdown banner, testimonials, FAQ accordion, logo list, multicolumn value props — 18+ sections, all JSON-configurable in the theme editor
- Storefront filtering and sorting on collections, related products, blog and article templates
- Speed-first: vanilla JavaScript, no jQuery, no frameworks; accessible drawers and modals (focus traps, Escape to close, reduced-motion respected)
- Colours, typography, radius and layout width all editable in theme settings — no code
- 100% original code (not a Dawn fork) — licensed for sale on any marketplace
