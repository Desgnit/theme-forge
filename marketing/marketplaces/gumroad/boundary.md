## Summary (one-liner)

A complete cricket club website — fixtures with proper scorelines, league table, squad averages, junior section and a first-ball countdown, in classic cricket green & gold.

## Description

Your club's website shouldn't look like a football template with the badge swapped. Boundary is built for how cricket clubs actually run — the demo club (Longmead CC, est. 1894, home at The Meadow) reads like a real village club, not lorem ipsum.

- **Match centre**: fixtures and results with real cricket scorelines (187/6 dec, 40-over formats), W/L/D/T badges, and a league table with your club's row highlighted — all plain HTML, edited by copy-paste
- **First-ball countdown** on the homepage (set one date attribute)
- **Squad grid** with batting and bowling roles, player pages with averages, and a "sponsor this player" revenue CTA
- Junior cricket page: All Stars and Dynamos age groups, coaching nights, parents' FAQ
- Match report layout, club history and honours, contact with ground directions
- 8 pages, HTML + compiled Tailwind CSS, no build step, vanilla JS, WCAG-AA patterns, tested at iPhone widths
- Every colour is a CSS variable — matching your exact club colours is a five-minute job, documented in the README

Perfect for: village and league cricket clubs, junior sections, club volunteers who got handed "the website job".
