## Description

Repset gives an independent gym a site that looks like the gym feels — charcoal, crimson and full-bleed training photography.

- **Full Site Editing block theme**: every layout is a pattern you edit visually in the WordPress Site Editor — no page builder plugins
- Ready-made patterns: cinematic hero, weekly class timetable grid, three-tier membership pricing table (highlighted middle tier), trainers grid, stats band, testimonials, opening-hours strip, CTA band
- Complete demo content for a believable gym (classes, coaches, memberships) — replace the words, keep the bones
- Oswald display type at poster sizes over dark image overlays; crimson accent system defined in theme.json
- Licensed demo photography included (Unsplash License)
- Works with any modern WordPress (6.x); GPL-licensed theme code as WordPress requires

Install: Appearance → Themes → Add New → Upload zip. Setup notes in readme.txt.
