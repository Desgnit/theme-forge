## Excerpt (short description)

A warm website for a village hall — hire rates and booking, a what's-on grid of weekly clubs, facilities and access details, run-by-volunteers friendly.

## Description

Thousands of village halls run on a volunteer with a spare evening — Commons is built for exactly that (the demo is Elmsworth Village Hall, a charity-run hall built 1928).

- **Hire the Hall** page: rooms with capacities and an hourly-rates table, resident and regular-club discounts, how booking and deposits work
- **What's On** grid: weekly clubs from pilates to short mat bowls, plus one-off events
- Facilities up front: capacity, kitchen, stage, parking, wheelchair access, hearing loop
- About page: the hall's history, the trustees, the charity number, how to volunteer
- Booking enquiry form with event type, date, room and numbers
- 5 pages, HTML + compiled Tailwind CSS, no build step, vanilla JS, WCAG-AA, iPhone-tested
- Every colour is a CSS variable — five-minute rebrand, documented in the README

Perfect for: village halls, community centres, church halls, scout huts, parish councils.
