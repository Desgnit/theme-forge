Brutal gym-poster block theme for independent gyms, strength clubs and functional fitness boxes.

== Description ==

Repset is a full site editing (FSE) block theme built for gyms that take training seriously. Deep charcoal backgrounds, a crimson accent that hits like a PB bell, concrete-grey supporting text and huge uppercase Oswald headlines give every page real poster energy.

The theme ships with a complete, ready-to-launch demo for a fictional strength gym ("Forge House Gym, Manchester") and eight production-grade block patterns:

* Hero — full-bleed cover with dark overlay and colossal headline
* Opening Hours Strip — slim band with weekday/weekend hours
* Stats Band — oversized numbers on a crimson gradient
* Class Timetable Grid — five image-led programme cards with weekly times
* Membership Pricing Table — three tiers with the middle tier highlighted
* Trainers Grid — coach portrait cards with credentials
* Testimonials — split image/quote section
* CTA Band — full-bleed call to action over a team photo

Typography pairs Oswald (500/600/700) for display with Inter (400/500) for body copy, loaded from Google Fonts with fluid, clamp-based sizing.

== Installation ==

1. In your WordPress admin, go to Appearance → Themes → Add New → Upload Theme.
2. Choose the repset.zip file and click Install Now.
3. Click Activate. The front page template renders the full demo layout immediately.
4. (Optional) Set a static front page under Settings → Reading if you want a different page to use the front-page layout.

== Editing patterns in the Site Editor ==

1. Go to Appearance → Editor (the Site Editor).
2. Open Templates → Front Page to edit the demo layout, or Patterns to browse the "Repset" category.
3. Click any section (hero, timetable, pricing…) and edit text, images, colors and links directly — every section is built from core blocks.
4. To reuse a section elsewhere, open the block inserter, search the "Repset" pattern category and drop it into any page or template.
5. Header and footer live under Patterns → Template Parts (or Appearance → Editor → Patterns → Header/Footer).

Colors, type scale and spacing are all defined in theme.json, so Styles (the half-moon icon in the Site Editor) lets you retheme everything globally.

== Frequently Asked Questions ==

= Does the theme require any plugins? =
No. Everything is core blocks.

= How do I change the accent color? =
Open the Site Editor → Styles → Colors and change "Crimson". Every button, rule and highlight follows it.
